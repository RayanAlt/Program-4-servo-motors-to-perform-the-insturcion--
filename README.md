# Program-4-servo-motors-to-perform-the-insturcion--
Arduino project that controls 4 servo motors and moves them to different angles.

This project contorls 4 servo motors using an Arduino 

The 4 servo motors move to different angles using the Arduino Servo library.

components of this Project

1 - Arduino 
2 - 4 Servo motors
3 - Jumper Wires

Servo Pin that i used in this project 

Motor1 = pin3
Motor2 = pin3
Motor3 = pin3
Motor4 = pin3
